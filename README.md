## 사용한 기술 스택
- Spring Boot 3
- Spring Security
- Spring JPA
- Java
- MySQL
- Thymeleaf
- LomBok
- HTML
- CSS
- Javascript

## 구현한 기능
- 스프링 시큐리티를 이용한 로그인 기능
- 스프링 시큐리티를 이용한 회원가입 기능
- 게시판 리스트
- 게시판 디테일 (다음글, 이전글 링크 구현)
- 게시글 생성 & 다중 파일 업로드 기능 구현
- 파일 다운로드 기능
- 게시글 수정
- 검색 기능
- 페이지 네이션 기능
- 댓글 생성
- 댓글 삭제
